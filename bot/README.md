# 🤖 بوت تداول كريبتو تعليمي (Binance)

بوت بسيط وآمن للتعلّم، يتداول على Binance باستراتيجية **تقاطع المتوسّطات
المتحرّكة (SMA crossover)**:

- عندما يتقاطع المتوسّط **السريع** صعودًا فوق **البطيء** → **شراء**.
- عندما يتقاطع هبوطًا تحته → **بيع** (إغلاق الصفقة).
- وقف خسارة اختياري يغلق الصفقة إذا هبط السعر كثيرًا.

> ⚠️ **تنبيه:** هذا أداة **تعليمية وليست نصيحة مالية**. بوتات التداول قد
> تخسر مالًا. ابدأ دائمًا بوضع المحاكاة، ثم الشبكة التجريبية، ولا تخاطر
> إلا بما تتحمّل خسارته.

---

## أفضل الممارسات (اقرأها قبل أي شيء)

1. **ابدأ بالمحاكاة ثم التجريبي ثم الحقيقي** — لا تقفز إلى أموال حقيقية.
2. **مفتاح API بصلاحية «تداول» فقط** — **عطّل السحب (Withdrawals) دائمًا**،
   وفعّل **تقييد عنوان IP** على المفتاح.
3. **لا تضع المفتاح في الكود أبدًا** — يُقرأ من ملف `.env` المتجاهَل من Git.
4. **خاطر بمبلغ صغير** لكل صفقة (`QUOTE_PER_TRADE`) بما تتحمّل خسارته.
5. **استخدم وقف الخسارة** (`STOP_LOSS_PCT`) للحدّ من الخسائر الكبيرة.
6. **لا توجد استراتيجية تربح دائمًا** — اختبرها طويلًا على التجريبي أولًا،
   وراقب النتائج في `trades.csv`.
7. **احذر الرافعة المالية** — هذا البوت يتداول **Spot فقط** (بلا رافعة) عمدًا.
8. **شغّله على جهاز يبقى يعمل** (حاسوبك أو خادم صغير) إن أردته يعمل باستمرار.

## خطوات التشغيل

### 1) المتطلبات
ثبّت Python 3، ثم المكتبات:
```bash
pip install -r bot/requirements.txt
```

### 2) الإعداد
انسخ ملف الإعداد واملأه:
```bash
cp bot/config.example.env bot/.env
```
عدّل `bot/.env` حسب رغبتك (العملة، المتوسّطات، المبلغ…). يبقى `BOT_MODE=dryrun`
في البداية.

### 3) المحاكاة (بلا مخاطرة — يُنصح بها أولًا)
```bash
python bot/trading_bot.py
```
سيطبع السعر والإشارات و**ماذا كان سيفعل** دون تنفيذ أي أمر حقيقي.
للتشغيل مرّة واحدة فقط (بدل التكرار): `python bot/trading_bot.py --once`

### 4) الشبكة التجريبية (أموال وهمية)
1. ادخل **https://testnet.binance.vision** وسجّل الدخول بحساب GitHub،
   ثم أنشئ **مفتاح API تجريبيًا** (HMAC).
2. ضع المفتاح والسر في `bot/.env`، وغيّر:
   ```
   BOT_MODE=testnet
   ```
3. شغّل البوت — الآن ينفّذ أوامر حقيقية بأموال **وهمية** على التجريبي.

### 5) الحقيقي (أموال حقيقية — بحذر شديد)
بعد أن تطمئن للنتائج على التجريبي:
1. أنشئ مفتاح API من حسابك الحقيقي على Binance: **صلاحية «Spot Trading» فقط،
   السحب مُعطّل، مع تقييد IP**.
2. ضعه في `bot/.env` وغيّر:
   ```
   BOT_MODE=live
   ```
3. شغّله بمبلغ صغير جدًا أولًا، وراقبه عن قرب.

## الإعدادات (في `bot/.env`)

| المتغيّر | المعنى | مثال |
|---------|--------|------|
| `BOT_MODE` | `dryrun` / `testnet` / `live` | `dryrun` |
| `SYMBOL` | زوج التداول | `BTCUSDT` |
| `INTERVAL` | إطار الشموع | `1h` |
| `FAST` / `SLOW` | فترتا المتوسّطين (سريع/بطيء) | `7` / `25` |
| `QUOTE_PER_TRADE` | المبلغ المُنفَق لكل شراء | `15` |
| `POLL_SECONDS` | كل كم ثانية يُعيد الفحص | `60` |
| `STOP_LOSS_PCT` | وقف الخسارة % (`0` = معطّل) | `5` |

## الملفّات
- `state.json` — حالة الصفقة الحالية (يُنشأ تلقائيًا، متجاهَل من Git).
- `trades.csv` — سجلّ كل صفقة (وقت، سعر، كمية، قيمة) لمراجعة الأداء.

## كيف توقفه
اضغط `Ctrl + C`. إذا كانت لديك صفقة مفتوحة فلن تُغلق تلقائيًا — أغلقها يدويًا
من Binance إن أردت، أو أعد تشغيل البوت ليُكمل إدارتها.
