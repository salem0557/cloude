# 🤖 بوت تداول كريبتو يتعلّم ويحدّث نفسه (Binance Spot)

بوت تداول متعدّد العملات على Binance، **يحدّث استراتيجيته تلقائيًا كل ساعتين**
ليبقى متوافقًا مع حركة السوق الأخيرة، ويعرض أداءه في لوحة على الموقع.

> ⚠️ **تنبيه مهم:** هذه أداة **تعليمية وليست نصيحة مالية**. بوتات التداول قد
> **تخسر مالًا**. ابدأ دائمًا بوضع المحاكاة، ثم التجريبي، ولا تخاطر إلا بما
> تتحمّل خسارته. أنت وحدك المسؤول عن أي تداول حقيقي.

---

## كيف «يتعلّم ويحدّث نفسه»؟

كل `OPTIMIZE_HOURS` (افتراضيًا **ساعتان**) ينفّذ البوت **تحديثًا ذاتيًا**:

1. **ضبط ذاتي بالباك-تيست** (`optimizer.py`) — يجرّب عشرات التركيبات من
   إعدادات الاستراتيجية (المتوسّطان السريع/البطيء، حدّ RSI، وقف الخسارة، جني
   الأرباح) على آخر بيانات السوق، ويختار **أفضل تركيبة ربحت** لكل عملة.
2. **نموذج تعلّم آلي (ML)** (`ml_model.py`) — يدرّب مصنّفًا صغيرًا
   (Gradient Boosting) لكل عملة يتنبّأ باحتمال صعود السعر، ويُستخدم
   **كتأكيد للدخول**: لا يُسمح بالشراء إلا إذا اتّفقت الإشارة الفنية **مع**
   ترجيح النموذج للصعود.
3. **اختيار الأفضل** — يرتّب كل العملات حسب نتيجة الباك-تيست ويتداول فقط
   أقوى `TOP_N` منها (والباقي تحت المراقبة).

> ملاحظة: مكتبات ML اختيارية. لو لم تُثبَّت، يعمل البوت بالإشارة الفنية وحدها
> ويتخطّى فلتر ML تلقائيًا — فيشتغل في كل مكان، ويصير «أذكى» عند توفّرها.

---

## الاستراتيجية باختصار

- **شراء** عندما يتقاطع المتوسّط السريع صعودًا فوق البطيء، والسوق ليس في تشبّع
  شرائي (RSI تحت الحدّ)، **ويؤكّد** نموذج ML الصعود.
- **بيع** عند تقاطع هبوطي، أو بلوغ **وقف الخسارة** أو **جني الأرباح**.
- تداول **Spot فقط** (بلا رافعة) عمدًا، مع حدود مخاطرة.

---

## أفضل ممارسات الأمان (اقرأها أولًا)

1. **محاكاة → تجريبي → حقيقي**؛ لا تقفز للأموال الحقيقية.
2. مفتاح API بصلاحية **«تداول» فقط**، مع **تعطيل السحب (Withdrawals)** و**تقييد IP**.
3. لا تضع المفتاح في الكود — يُقرأ من `bot/.env` المتجاهَل من Git.
4. خاطر بمبلغ صغير لكل صفقة (`QUOTE_PER_TRADE`)، وفعّل `DAILY_LOSS_LIMIT`.
5. الوضع الحقيقي **مقفول** بمتغيّر تأكيد صريح (انظر أدناه).

## التشغيل السريع

```bash
pip install -r bot/requirements.txt
cp bot/config.example.env bot/.env      # ثم عدّله
python bot/trading_bot.py               # محاكاة (آمن)
python bot/trading_bot.py --once        # دورة واحدة فقط
```

### الوضع التجريبي (أموال وهمية)
1. أنشئ مفتاح API من <https://testnet.binance.vision>.
2. في `bot/.env`: ضع المفتاح والسر و`BOT_MODE=testnet`.

### الوضع الحقيقي (أموال حقيقية — بحذر شديد) 🔴
1. مفتاح API حقيقي: **Spot Trading فقط، السحب معطّل، IP مقيّد**.
2. في `bot/.env`:
   ```
   BOT_MODE=live
   CONFIRM_LIVE=I_UNDERSTAND_THE_RISK
   BINANCE_API_KEY=...
   BINANCE_API_SECRET=...
   ```
   البوت **يرفض الإقلاع** في الوضع الحقيقي بدون سطر `CONFIRM_LIVE` بالضبط.
3. ابدأ بمبلغ صغير جدًا وراقبه عن قرب.

## لوحة العرض

أثناء عمله يكتب البوت لقطة حالة (بلا أي مفاتيح) إلى
`docs/crypto/data/bot.json`، وتعرضها صفحة **`docs/crypto/bot.html`**: الرصيد،
الصفقات المفتوحة، الإعدادات المتعلّمة لكل عملة، نتائج الباك-تيست، ودقّة ML،
وآخر الصفقات. تُحدَّث الصفحة كل 30 ثانية.

> لإظهار البيانات على الموقع المنشور (GitHub Pages) لا على الخادم فقط، اجعل
> خادمك يرفع `bot.json` للمستودع دوريًا (commit/push)، أو افتح اللوحة محليًا.

## الإعدادات (`bot/.env`)

| المتغيّر | المعنى | مثال |
|---------|--------|------|
| `BOT_MODE` | `dryrun` / `testnet` / `live` | `dryrun` |
| `CONFIRM_LIVE` | إجباري للوضع الحقيقي | `I_UNDERSTAND_THE_RISK` |
| `SYMBOLS` | سلّة العملات | `BTCUSDT,ETHUSDT,…` |
| `TOP_N` | كم عملة يتداول من السلّة | `3` |
| `INTERVAL` | إطار الشموع | `1h` |
| `HISTORY` | عدد الشموع للباك-تيست/ML | `500` |
| `QUOTE_PER_TRADE` | المبلغ لكل شراء | `15` |
| `MAX_OPEN_POSITIONS` | أقصى صفقات مفتوحة | `3` |
| `DAILY_LOSS_LIMIT` | حدّ خسارة يومي (`0`=معطّل) | `50` |
| `POLL_SECONDS` | كل كم ثانية يفحص السوق | `60` |
| `OPTIMIZE_HOURS` | كل كم ساعة يحدّث نفسه | `2` |
| `PAPER_EQUITY` | رصيد بدء المحاكاة فقط | `1000` |

## الملفّات
| الملف | الوظيفة |
|------|---------|
| `trading_bot.py` | المنسّق الرئيسي وحلقة التشغيل |
| `exchange.py` | الاتصال بـ Binance (محاكاة/تجريبي/حقيقي) |
| `strategy.py` | توليد الإشارات (SMA + RSI) |
| `indicators.py` | المؤشّرات الفنية |
| `backtest.py` | محرّك الاختبار التاريخي |
| `optimizer.py` | الضبط الذاتي (grid-search) |
| `ml_model.py` | تدريب/تنبؤ نموذج ML |
| `dashboard.py` | كتابة بيانات اللوحة |
| `state.json` | حالة الصفقات (يُنشأ تلقائيًا، متجاهَل) |
| `trades.csv` | سجلّ كل الصفقات لمراجعة الأداء |
